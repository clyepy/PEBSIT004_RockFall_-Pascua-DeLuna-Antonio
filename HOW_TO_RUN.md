# HOW TO RUN – ROCKFALL

This guide provides exact steps for opening, building, and running the Rockfall project.

---

## ▶ Running the Game (Build Version)
1. Download the provided **Rockfall.zip**.
2. Right-click → **Extract All**.
3. Open the extracted folder.
4. Double-click:
   - **Rockfall.exe** (Windows 64-bit)
5. The game starts immediately.

---

## ▶ Opening the Project in Unity 
1. Install **Unity Hub**.
2. Install **Editor** using Unity Hub.
3. Open Unity Hub → **Projects** tab.
4. Click **Add**.
5. Select the **Rockfall** project folder.
6. Once added, click **Open**.
7. Go to:
   - `Assets/Scenes/MainScene.unity`
8. Press **Play** to run the game in the editor.

---

## ▶ Building the Project (If Needed)
1. In Unity, go to **File → Build Settings**.
2. Select **PC, Mac & Linux Standalone**.
3. Choose **Windows x86_64**.
4. Click **Add Open Scenes**.
5. Press **Build**.
6. Choose output folder → Unity generates the `.exe` build.

---

## ▶ Minimum System Requirements

### **For Running the Build**
- OS: Windows 10 / 11  
- CPU: Intel i3 or equivalent  
- RAM: 4 GB minimum  
- GPU: Integrated graphics supported (Intel HD 4000 or newer)  
- Storage: 500 MB free space  

### **For Opening in Unity**
- RAM: 8 GB (16 GB recommended)  
- CPU: Intel i5 / Ryzen 5 or higher  
- GPU: Any GPU supporting Unity Editor  
- Unity Version: **6000.2.2f1**
