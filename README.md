# ROCKFALL

Rockfall is an arcade-style space survival game where the player pilots a ship while avoiding falling asteroids, shoot asteroid that are moving close to the spacestation and surviving as long as possible. Built in Unity, the game features responsive controls, dynamic hazards, and a simple scoring system based on time and survivability.

## Engine / Language / Version
- **Engine:** Unity (LTS)
- **Language:** C#
- **Platform:** PC (Windows)

## Controls
**Mobile (On-Screen Controls):**
- **Move:** Drag your thumb inside the on-screen joystick located at the lower-left of the screen.
- **Shoot:** Tap or hold the on-screen shoot button at the lower-right.
- **Pause:** Tap the pause icon in the upper-right corner.


## How to Run (Quick Guide)
1. Download the `Rockfall.zip` file.
2. Extract the folder to any directory.
3. Open the folder and double-click:
   - **Rockfall.exe** (Windows)
4. Play the game.

- Camera shake effects may vary depending on device performance.
